# The Flow English School

Site profissional da The Flow English School - escola de inglés com foco em conversação real, fluência prática e aprendizado natural para jovens e adultos brasileiros.

## Características

- Inglés vivo e conversacional
- Design moderno e mobile first
- Funil de conversão otimizado
- Copywriting persuasivo e humano
- Integração com WhatsApp

## Como acessar

O site está publicado em: https://suportebyenglish-lab.github.io/the-flow-english-school/

## Arquivos do projeto

- `index.html` - Página principal (HTML5 semântico)
- `styles.css` - Estilos CSS3 (paleta azul petróleo + dourado)
- `LICENSE` - Licença MIT

## Desenvolvido com

- HTML5 (semântico)
- CSS3 (Grid, Flexbox, Responsive)
- JavaScript (interatividade leve)
- Mobile First
